SPASE Base Model

The SPASE metadata model can be expressed in many forms. One of those forms is as an XML Schema. An XML Schema is a machine readable form (see schemaView the SPASE XML Schema). Here you can find human readable documentation which includes standard model diagrams.

Each version of the metadata model is fully documented. The available versions are available at: http://www.spase-group.org/data/model/View documentation on the SPASE Information Model.
