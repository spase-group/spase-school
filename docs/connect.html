SPASE Mailing Lists

The Space Physics Archive Search and Extract (SPASE) consortium is an international community of scientists, specialists, information engineers and system designers who are endeavouring to create a standards and services to enable the open exchange of Heliophysics data.

SPASE General Discussions

There main SPASE mailing is at: 

    spase@listserv.gsfc.nasa.gov
To subscribe send an e-mail to spase@listserv.gsfc.nasa.gov with
  • Subject line blank.
  • a message body of: 
          subscribe spase your_email_address
Anyone is welcome to join. This is also where data model changes are first posted. There is a telecon held about twice a month. The times are announced on the "spase" Mailing list. Everyone can join the telecon and participate.

SPASE Metadata Working Group (SMWG)

The SPASE metadata working group discusses implementation issues, mostly focused on the generation of metadata and maintaining a common top level registry. Its maintained at: 

    http://groups.google.com/group/vxo-spase-smwg?hl=en
email: vxo-spase-smwg@googlegroups.com

SPASE-QL

SPASEQL is an effort to use the SPASE data model to create a standardized communication language for Virtual Observatories. This group creates standards for query and response message construction and builds tools to implement SPASEQL. 

   http://groups.google.com/group/hdmc-spaseql?hl=en&lnk=
email: hdmc-spaseql@googlegroups.com

HDMC Mailing Lists

The Heliophysics Data and Model Consortium (HDMC) is a NASA group designed to coordinate the activities of NASA funded projects that are part of establishing an integrated Heliphysics Data Environment (HPDE).

HDMC General Discussions

Group to provide content of general interest and a common location for information about the NASA Heliophysics Data and Model Consortium 

   http://groups.google.com/group/hdmc?hl=en&lnk=
email: hdmc@googlegroups.com

HDMC Analytics

This forum is for the discussion of analytics issues among members of the HDMC (Heliophysics Data Model Consortium). 

   http://groups.google.com/group/hdmc-analytics?hl=en&lnk=
email: hdmc-analytics@googlegroups.com

HDMC Eventlists

Scientists currently manually generate, manage and share lists of observations. The goal of this group is to improve list querying and enable annotating and intersecting lists of events across a full range of Heliophysics data sources, and leverage the services of the VxOs and SPDF. 

   http://groups.google.com/group/hdmc-eventlists?hl=en&lnk=
email: hdmc-eventlists@googlegroups.com

HDMC Data Access

A forum for discussing techniques and tools to deal with diverse science data formats in the Heliophysics community. The ultimate aim is to jointly create easy-to-use mechanisms for obtaining diverse, distributed data in a format that is immediately useful for scientific analysis. 

   http://groups.google.com/group/hdmc-dataaccess?hl=en&lnk=
email: hdmc-dataaccess@googlegroups.com
