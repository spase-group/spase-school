Validation

Top

Validation of a resource description ensures that it complies to an instance of the SPASE data model. While the SPASE data model is relatively stable, there are new versions released from time-to-time. Each version has a prescribed set of elements for each resource class.

In addition to conforming to a specific version of SPASE data model, many a resource descriptions contain references to other resources through resource identifiers and to stored resources using a URL. A full validation ensures the integrity of these references.

SPASE provides both an on-line validation tool and a command-line validation tool as part of the SPASE Resource Tools.
